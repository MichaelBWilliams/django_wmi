About

The Web Map Interface downloads tool provides ready-to-access map atlases and individual administrative area maps for important thematic data available in each country. The individual country pages display the available mapping themes, which have been selected as relevant to the MFI and include links for downloading each of these map sets. While the number of individual maps availabe for each country varies, the entire VisionFund map catalog includes nearly 2,000 maps. If you would like to see more information about the data used to generate these maps or request additional mapping related to your MFI, please see the Data Links page or go to the Contact page.

How to use the map downloads tool

The download interface is organized by the country or countries available to the user. These are displayed on the panel to the left. To navigate to a specific download page, click the name of the country. *Not all users may see the same selections.

Once a country has been selected, all map atlases available for that country will be displayed on the main page. To view download options for a specific theme, click on the thumbnail image of the map.

After making a selection a window will open to show information about the theme (1) and download options.

(2) Map atlases can be downloaded as a complete package, formatted in a single .pdf file or a .zip file of .jpeg images. These files can be large and will take a longer time to download depending on the available internet connection speeds

(3) Administrative units are available for individual download as .jpeg files. These smaller files are the same as those within the complete map packages above, but can be accessed idependently.

Map Themes

The following map themes have been used to create maps and map packages for each country within the project scope. Some themes are only available for particular regions and not all countries will have the same map packages available.

Drive time to Nearest Branch

The driving distance atlas displays the estimated driving distance a client or loan officer would have to travel to reach the nearest branch office. This has been calculated using branch location coordinates in conjunction with available road network data. This analysis is meant to branch network coverage within a country, as well as provide estimations of driving distances to any area within the network.

Landcover

The landcover atlas displays multiple landcover classes defined and calculated by the European Space Agency Climate Change Initiative (ESA-CCI). MFI branch locations are shown overlayed onto these classes. To see the area analysis completed for the cropland class, please see the Croplands atlas. This mapping is meant to provide a comprehensive view of the landcover types within an MFI's country to assist with planning and operations.

Croplands

The croplands atlas diplays three different cropland-related landcover classes from the European Space Agency Climate Change Initiative (ESA-CCI) dataset. MFI branch locations are overlayed onto these classes and cropland area statistics are calculated by administrative unit. This mapping is meant to provide an easy to discern view of the croplands within a country to assist with planning and operations.
